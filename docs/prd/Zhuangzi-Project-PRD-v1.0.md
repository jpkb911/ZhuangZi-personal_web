## 1. 概述 (Overview)

### 1.1 项目背景
本项目旨在构建一个以中国古代哲学家庄子为主题的个人网站。通过极简的水墨设计风格和沉浸式的阅读体验，全方位展示庄子的生平事迹与哲学著作，让用户在数字化空间中感受“逍遥”与“道”的韵味。

### 1.2 核心目标
1.  **意境传达**：通过视觉设计（留白、水墨、视差）传达庄子哲学的空灵与深邃。
2.  **知识普及**：清晰展示庄子生平，提供《庄子》全集的原文、译文及深度解析。
3.  **思想共鸣**：引导用户在阅读中产生思考，提供哲思卡片便于感悟与传播。

### 1.3 用户角色
| 角色 | 描述 | 核心诉求 |
| :--- | :--- | :--- |
| **访客/读者** | 对中国传统文化或哲学感兴趣的用户 | 了解庄子生平，阅读经典篇章，寻求精神共鸣。 |

---

## 2. 核心用户旅程 (User Journey Map)

### 2.1 阶段划分
我们定义的**核心用户旅程**分为以下几个阶段：

1.  **初识逍遥（首页）**：用户进入网站，被视觉氛围吸引，建立第一印象。
2.  **知人论世（生平）**：深入了解庄子的时代背景与个人经历。
3.  **品读经典（作品）**：沉浸式阅读《庄子》文本，查阅注解与译文。
4.  **思想共鸣（感悟）**：在阅读后获取哲学洞察，进行思考或分享。

### 2.2 核心流程图 (Mermaid)
```mermaid
graph TD
    A[用户进入网站] --> B[首页: 初识逍遥]
    B -->|极简导航| C{核心探索路径}
    
    C -->|知人论世| D[生平介绍页]
    D --> D1[总括简介]
    D --> D2[生平时间轴 (视差滚动)]
    D2 --> D2a[查看轶事: 濠梁之辩/鼓盆而歌]
    
    C -->|品读经典| E[作品全集页]
    E --> E1[目录选择: 内/外/杂篇]
    E1 --> E2[阅读详情页]
    E2 --> E2a[原文阅读 (悬浮注解)]
    E2 --> E2b[查看文末哲思卡片]
    
    B -->|每日一句| F[随机名言浮现]
```

## 3. 用户故事详细定义 (User Stories)

### 3.1 阶段一：初识逍遥（首页）

#### US-01: 首页浏览与氛围感知
| 属性 | 描述 |
| :--- | :--- |
| **简述** | 作为访客，我希望看到一个极简且富有水墨意境的首页，以便快速感受到庄子的哲学氛围并找到内容入口。 |
| **优先级** | P0 (Must Have) |

**业务规则与逻辑**：
1.  **视觉基调**：大面积留白，配合动态的水墨晕染背景（CSS/Canvas 实现渐变与流动感）。
2.  **核心元素**：
    *   **中心背景**：庄周梦蝶的水墨/半透明图像，居中且位于文字底层（低透明度），营造“梦与觉”的交织感。
    *   **Logo**：极简的“庄”字印章或书法体。
    *   **Slogan**：竖排大字“独与天地精神往来”，叠加在梦蝶图像之上。
    *   **每日一句**：从名言库中随机抽取一句，以淡入淡出动画浮现在屏幕的随机空白位置（避免遮挡核心内容）。
3.  **导航设计**：隐藏式或极简文字链接（生平、作品、搜索），不破坏画面整体感。

**验收标准 (Acceptance Criteria)**：
- [ ] 进入首页时，背景应有流畅的水墨晕染动画效果。

- [ ] 每日一句每次刷新或每隔一定时间（如 30s）自动更换，且位置随机但不重叠 Slogan。
- [ ] 导航链接清晰可用，点击可跳转至对应页面。
- [ ] 移动端适配：竖排文字在窄屏下保持可读性，背景动画不卡顿。

### 3.2 阶段二：知人论世（生平）

#### US-02: 生平时间轴与视差滚动
| 属性 | 描述 |
| :--- | :--- |
| **简述** | 作为访客，我希望通过时间轴和图文互动的形式浏览庄子生平，以便深入了解他的生命历程。 |
| **优先级** | P1 (Should Have) |

**业务规则与逻辑**：
1.  **叙事结构**：
    *   **顶部**：一段总括性简介（名周，宋国蒙人...）。
    *   **主体**：垂直时间轴，串联关键节点（出生、漆园吏、拒楚相、濠梁观鱼、妻死鼓盆、逝世）。
2.  **视差效果**：
    *   背景层（山水长卷）滚动速度慢于前景层（文字/时间节点），营造空间深度的视差感。
3.  **插图展示**：关键节点（如濠梁之辩）配以水墨风格插画，随滚动渐显。

- [ ] 关键插图在进入视口时有淡入或浮现动画。
- [ ] 文本内容准确无误，符合史料记载。

### 3.3 阶段三：品读经典（作品）

#### US-03: 沉浸式阅读与注解
| 属性 | 描述 |
| :--- | :--- |
| **简述** | 作为访客，我希望阅读《庄子》原文并能方便地查看注解和哲思解析，以便深入理解其思想。 |
| **优先级** | P0 (Must Have) |

**业务规则与逻辑**：
1.  **目录导航**：支持按内篇、外篇、杂篇筛选和跳转。
2.  **正文阅读**：
    *   大字号，衬线体（宋体/楷体），排版疏朗。
    *   支持夜间模式切换。
3.  **悬浮注解**：
    *   原文中的生僻字或关键概念（如“鲲”、“鹏”、“野马”）下方标示虚线或高亮。
    *   点击/悬停时，在旁边或上方弹出气泡显示注音和释义。
4.  **文末哲思**：
    *   每篇结束后，展示一个设计精美的“哲思卡片”，包含核心思想总结、名家点评摘录。

**UI/UX 线框图 (ASCII)**：
```
+---------------------------------------------------------------+
|  [< 目录]       逍遥游 · 内篇 第一       [Aa 字体] [🌙 夜间]  |
+---------------------------------------------------------------+
|                                                               |
|  北冥有鱼，其名为鲲。鲲之大，不知其几千里也...                |
|  (正文区：大字号，衬线体，排版疏朗)                           |
|                                                               |
|  化而为鸟，其名为[鹏]*。鹏之背，不知其几千里也...             |
|                  ^                                            |
|                  | (点击/悬停 [鹏] 字)                        |
|        +-------------------------+                            |
|        | [注] 鹏：传说中最大的鸟。 |                            |
|        | 喻指超越世俗的伟大境界。  |                            |
|        +-------------------------+                            |
|                                                               |
|  (正文结束)                                                   |
|                                                               |
|  +-------------------------------------------------------+    |
|  |  💡 哲思共鸣：绝对自由的境界                          |    |
|  +-------------------------------------------------------+    |
|  |  庄子通过鲲鹏之变，阐述了从小大之辩到无待之游...      |    |
|  +-------------------------------------------------------+    |
+---------------------------------------------------------------+
```

**验收标准 (Acceptance Criteria)**：
- [ ] 目录导航准确，能正确跳转至对应篇章。
- [ ] 点击难字/关键词能准确弹出注解气泡，再次点击或点击空白处关闭。
- [ ] 夜间模式切换顺畅，文字颜色与背景对比度符合无障碍标准。
- [ ] 文末哲思卡片样式美观，内容与篇章主题契合。

## 4. 非功能性需求 (Non-functional Requirements)

1.  **性能**：首页首屏加载时间 < 1.5s（优化图片和 Canvas 脚本）。
2.  **兼容性**：支持主流现代浏览器（Chrome, Safari, Edge, Firefox）及移动端浏览器。
3.  **SEO**：对《庄子》篇章内容进行 SEO 优化，确保相关关键词可被搜索引擎检索。
4.  **可访问性**：字体大小可调，色彩对比度适宜，支持屏幕阅读器。

---

## 5. 数据与内容需求
- **文本库**：《庄子》全集（内7+外15+杂11）的原文、校对版注音及释义。
- **名言库**：精选 50+ 条庄子名言用于首页随机展示。
- **素材库**：水墨风格背景图、生平关键节点插画。
